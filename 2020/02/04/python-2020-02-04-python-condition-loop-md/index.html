조건문

특정 조건에 따라서, 코드를 실행하고자 할때 사용합니다.

if, else, elif

정반대의 조건 이면 else, 그게 아니면 elif를 사용합니다.
elif를 만족하면 조건문을 빠져 나옵니다. 즉 아래쪽의 코드를 실행하지 않습니다.

반복문
