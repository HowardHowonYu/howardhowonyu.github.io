분산과 표준편차

Posted on 2020-02-07 Edited on 2020-03-02

확률분포의 분산(variation)

  • Var[X]

분산의 성질

  • Var[X] >= 0
  • Var[C] = 0
  • Var[cX] = c**2Var[X]
  • Var[X] = E[X**2] - m**2
  • Var[X+Y] = Var[X] + Var[Y] + 2E[(X-m)(Y-m)]

확률변수의 독립

  • 두 확률변수가 서로에게 영향을 미치지 않는다
  • E[(X-m)(Y-m)] = 0
  • Var[X+Y] = Var[X] + Var[Y]

## 표본분산의 기대값

비대칭도와 첨도

  • 비대칭도는 3차 모멘트 , 첨도는 4차 모멘트

모멘트

  • 기대값, 분산은 확률분포의 모멘트
  • E[Y] = E[X]가 같고 여러 제곱들이 계속 같으면
  • X와 Y는 같은 확률분포

다변수 확률변수

이산확률변수 에서 다변수 확률변수

  • 조건부확률질량함수
